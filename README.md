# System Engineering & DevOps Project
A collection of shell scripts and exercises for learning system engineering and DevOps.

## Project Structure
- `0x00-shell_basics/` : beginner shell scripts
- `0x01-shell_permissions/` : exercises on file permissions
- `README.md` : this file

## Requirements
- Scripts must start with `#!/bin/bash`
- Scripts should follow the project-specific rules (2 lines, no forbidden syntax, etc.)

## Usage
Run scripts from the terminal. Example:
```bash
./0x00-shell_basics/your_script.sh
